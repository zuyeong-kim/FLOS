# 🌿 FLOS (플로스)

> 꽃으로 일상을 플러스하는 감성 플로리스트 브랜드 웹 랜딩페이지

---

## ✨ Overview

FLOS는 꽃을 단순한 상품이 아닌 감정과 일상을 연결하는 경험으로 재해석한 브랜드입니다.<br>

이 프로젝트는 FLOS 브랜드의 세계관을 시각적으로 전달하고,<br>
사용자가 자연스럽게 상품과 서비스 흐름을 이해할 수 있도록 설계된 반응형 랜딩페이지입니다.

---

## 🎯 Project Goal

- 감성 중심 브랜드 아이덴티티 전달
- 스크롤 기반 스토리텔링 구조 구현
- 상품/서비스 정보의 자연스러운 흐름 구성
- 시각 중심 UI/UX 설계 연습

---

## 🧩 Features

- 히어로 섹션 (브랜드 메시지 + 비주얼 임팩트)
- 컬렉션/상품 카드 UI
- 브랜드 가치 소개 섹션
- 정기구독 플랜 UI
- 스토리텔링형 이미지 섹션
- 아티스트/파트너 모집 영역
- FAQ 아코디언 구조
- CTA 및 푸터 구성
- 반응형 웹 디자인 (PC / Tablet / Mobile 대응)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap (Grid System 활용)
- JavaScript (interactive 요소 최소 적용)
- Swiper.js (슬라이드 구성 사용 시)
- Figma (UI/UX 디자인)

## 📌 Git Convention

### Commit Message

- feat : 새로운 기능 추가
- fix : 버그 수정
- style : 코드 스타일 변경 (기능 변경 없음)
- refactor : 코드 리팩토링
- design : UI / 디자인 변경
- docs : 문서 수정 (README 등)
- chore : 기타 작업 (빌드 설정, 패키지 등)

### Versioning

- MAJOR.MINOR.PATCH (예: 1.0.0)

- MAJOR: 큰 구조 변경
- MINOR: 기능 추가
- PATCH: 버그 및 오류 수정
